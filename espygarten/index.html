espygarten

use with std; to recieve the "standard library" from the runtime. with turns the rest of the scope into a function and returns it, which the runtime will immediately call with the library as an argument.

this library provides std.io.print, which can be used to append text to this page's "console output". you may invoke a function by following it with a value, for example: std.io.print "Hello, world!" calls std.io.print with "Hello, world!" as its argument. more information on this library may be found in the runtime interoperability section.

the value returned by your function will be displayed underneath the console output.

runtime interoperability

an espyscript program's host (espygarten, in this page's case) may provide custom structures to the program by passing them into a function as its argument. these structures may be used to augment espyscript with functionality it does not otherwise provide, such as io, string manipulation, or efficient data structures.

the libraries provided by espygarten are listed below.

io library

facilitates input and output on this page

io.print

appends the provided string to this page's "console output"

string library

provides basic operations for dynamic creation and manipulation of strings.

string.concat

expects a tuple of strings as an argument and concatenates them into a single string
