use with std; to recieve the "standard library" from the runtime. with turns the rest of the scope into a function and returns it, which the runtime will immediately call with the library as an argument.

this library provides std.io.print, which can be used to append text to this page's "console output". you may invoke a function by following it with a value, for example: std.io.print "Hello, world!" calls std.io.print with "Hello, world!" as its argument.

the value returned by your function will be displayed underneath the console output.
